# UI Rules — Startup Hunt

## Общий стиль

- Минималистичный dashboard
- Светлый фон (`#FAFAFA`), белые карточки/поверхности
- Акцент — оранжевый Product Hunt (`#FF6154`)
- Интерфейс на русском языке

## Layout

- Таблица стартапов — главный экран после логина (`/startups`)
- Navbar: логотип, ссылки по роли, имя пользователя, выход
- Контент: max-width 1200px, центрирован, padding 24px

## Формы

- Одна колонка, max-width 480px
- Label над полем, placeholder как подсказка
- Кнопка submit — primary (оранжевая), full-width на mobile
- Ошибки валидации — красный текст под полем

## Голосование

- Две иконки-кнопки (▲ / ▼) слева от числа рейтинга
- Активный голос подсвечен `--upvote` или `--downvote`
- Рейтинг — жирный, по центру между кнопками

## Таблица

- Колонки: Рейтинг | Название | Автор | Дата | Действия
- Строка кликабельна → детальная страница
- Hover — лёгкий фон `#F3F4F6`

## Mobile (< 768px)

- Таблица превращается в stack-карточки
- Без горизонтального скролла
- Navbar — hamburger или упрощённый

## Состояния

- **Loading:** skeleton-строки в таблице
- **Empty:** «Пока нет стартапов» + CTA для founder
- **Pending role:** «Ожидайте назначения роли администратором»
- **Error:** toast с описанием

## Доступность

- Кнопки голосования с `aria-label` («Голосовать за», «Голосовать против»)
- Focus ring на интерактивных элементах
